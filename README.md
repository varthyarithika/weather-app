# Weather App

A simple weather app built with HTML, CSS, and JavaScript.

## Features

- Search weather by city name
- Shows temperature, humidity, wind speed, and weather description
- Uses the Open-Meteo weather and geocoding APIs (no API key required)
- Displays friendly error messages for empty or invalid searches

## How to Use

1. Open `index.html` in a browser.
2. Search for a city.

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
