# Weather App

A simple weather app built with HTML, CSS, and JavaScript.

## Features

- Search weather by city name
- Shows temperature, humidity, wind speed, and weather description
- Uses the OpenWeatherMap API
- Displays friendly error messages for empty or invalid searches

## How to Use

1. Get a free API key from OpenWeatherMap.
2. Open `script.js`.
3. Replace `YOUR_API_KEY_HERE` with your API key.
4. Open `index.html` in a browser.

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
